# Conversion Funnel & User Retention Analysis

SQL-based analysis of e-commerce conversion funnels and user retention using cohort analysis and business insights.

---

## Project Overview

This project focuses on analyzing user behavior in an e-commerce conversion funnel and evaluating user retention through cohort analysis. The objective was to identify critical drop-off points, retention trends, and opportunities to improve customer experience and conversion performance.

---

## Objectives

- Analyze conversion rates across key funnel stages.
- Identify stages with the highest user drop-off.
- Evaluate user retention using cohort analysis (D7, D14, D21, D28).
- Detect behavioral patterns and generate actionable business insights.

---

## Tools & Technologies

- SQL
- CTEs
- JOINs
- GROUP BY
- Exploratory Data Analysis (EDA)
- Google Sheets

---

## Key Insights

- Identified critical drop-off points in the conversion funnel.
- Detected differences in conversion performance across countries.
- Analyzed user retention trends through cohort analysis.
- Proposed data-driven recommendations to improve conversion and retention.

---

## Business Impact

The analysis helped identify friction points in the purchasing process and opportunities to improve user retention and customer experience through data-driven decision-making.
